    • projects.
          • League of Legends Analytics
          • A statitical build / strategies analytics tool designed for League of Legends. The data mining and processing was done using Python. The API layer uses an Express server ontop of a Postgres database. The front end was built using React + Redux with custom Webpack configs.
          • Technology: Python, Postgres, Node.js, Express, React, Redux, Webpack
          • Pokedex
          • Extended and restructured version of my previous Pokedex. The front end for this Pokedex was built using React + Redux. The backend was implmented using an Express server built ontop of a custom Postgres database from scraped Pokemon data
          • Technology: Postgres, Node.js, Express, React, Redux, Webpack
          • Audio Playlist / Visualizer
          • Extended and restructured version of my previous audio visualizer. This visualizer comes with a playlist feature implemented using a flux based architecture. The geometric visualizations are rendered using raw HTML canvas drawing techniques similar to processing
          • Technology: WebAudio, React, flux, JavaScript, SCSS, HTML5, Webpack
          • D3 Geographic Esports Visualisation
          • Created a 3D globe choropleth (heat map) of esports earnings for selected competitive games. Features include a self-rotating globe, zoomable interface and a tooltip when a country is hovered over. The visualization accepts user input and changes based on the selected game.
          • Technology: D3.js, JavaScript, SCSS, HTML5, Gulp
          • D3 Audio Visualizer
          • Dynamically creates animated graphs of uploaded audio files. Graphs utilize frequency and time domain data passed through custom D3 components and then into the DOM as SVG elements. Experience 2 color themes and 3 graph types (radial, frequency, wave).
          • Technology: D3.js, JavaScript, Sass, HTML5, Gulp
          • React 3D League of Legends Champion Selector
          • A compilation of LoL Champion data and images scraped using Node.js and rendered with React.js. The unique interface utilizes 3D transform properties in CSS to give the selection pannel the illusion of perspective. Images are optimized with the use of gulp.
          • Technology: React.js, Node.js, Sass, HTML5, Gulp
          • AngularJS Pokedex
          • A highly interactive recreation of the classic digital Pokemon encyclopedia. I wrote a scrapper to collect the Pokemon data and mass download sprites and images. The interface is mobile compatible and was built using AngularJS. Responsive layout utilizes the CSS flexbox module.
          • Technology: AngularJS, Node.js, Sass, HTML5, Gulp
